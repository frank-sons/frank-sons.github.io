Agile Entwicklung

Agile Entwicklung ist heutzutage ein komplexes Umfeld, welches weit über die Einführung von einer neuen Projektmethode hinaus geht. Scrum, Kanban, XP usw. können dabei helfen eine passende Teamkultur zu etablieren, aber das alleine hilft noch nicht langfristig eine bessere Software und Code Qualität zu erreichen. Bei der Einführung und der Weiterbildung Ihres Teams in agilen best practices oder nachhaltiger Software-Architektur stehe ich Ihnen gerne zur Seite.

Hier finden Sie ein paar Themen, bei denen ich Ihnen mit workshops, Trainings und Unterstützung im Team helfen kann:

  •  Unit Testing
  •  Code Reviews
  •  Pair Programming
  •  Legacy code
  •  Refactoring
  •  Design Patterns

Continuous Workflows

Ein Thema welches in vielen Teams unterschätzt wird, ist Continuous Integration und Deployment. Agile Entwicklung wird oft eingeführt um schnellere Release Zyklen zu ermöglichen, aber es dann stellt sich nicht selten heraus, dass es mit der vorhandenen Code Basis gar nicht so einfach ist, oder der Deployment Prozess so komplex ist, dass es schwierig ist ihn überhaupt zu automatisieren.
Oft läuft zwar ein Continuous Integration System, aber es wird nicht aktiv genutzt. Das Team verlässt sich darauf, dass schon eine Mail kommt wenn etwas falsch ist oder der build gebrochen ist. Auch hier kann eine Schulung oder ein Workshop helfen, die Qualitätsmetriken, welche bei einem gut aufgesetzten System vorhanden sind, aktiv zu nutzen, um rechtzeitig Tendenzen in der Code Qualität zu erkennen.

Meine Leistungen:

  •  Aufsetzen von Continuous Integration workflows
  •  Workshops zu Qualitätsmetriken innerhalb der Continuous Integration workflows
  •  Unterstützung beim etablieren von Continuous Deployment/Delivery workflows

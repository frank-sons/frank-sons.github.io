Software & Code Qualität

Software & Code Qualität, oder auch externe und interne Qualität spielen beide eine wichtige Rolle für den Erfolg eines Projektes. Wie wichtig diese beiden Seiten sind, kann von Projekt zu Projekt variieren. Daher ist es wichtig, gemeinsam zu erarbeiten, welcher Fokus benötigt wird.
Auch im agile fluency model finden sich verschiedene Stufen wieder, bei denen die Qualität jeweils einen anderen Fokus hat. "Ein Stern"-Teams haben eine sehr großen Fokus auf der externen Qualität, "Zwei Sterne"-Teams legen einen verstärkten Fokus auf die interne, Code Qualität, um bei langfristigen Projekten und Produkten die Anzahl der Fehler zu verringern und die Produktivität hoch zu halten.
